# Contact List System

A simple Contact List Management System built using Python. This project allows users to add, view, search, update, and delete contacts using a CSV file.

## Features

- Add Contact
- View Contacts
- Search Contact
- Update Contact
- Delete Contact

## Technologies Used

- Python
- CSV File Handling

## Project Structure

```
Contact-List-System/
│── contact_list.py
│── contacts.csv
│── README.md
```

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/tejaborukati-hue/Contact-List-System-in-Python.git
```

2. Open the project folder.

3. Run the program:

```bash
python contact_list.py
```
